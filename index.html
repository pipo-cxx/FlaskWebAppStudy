How to run my WebApp

For running my WebApp you will need Python 3 and Flask installed. Also a graphical web browser (optional, but recommended).

First, clone the Github repository by using

https://github.com/pipo-cxx/FlaskWebAppStudy.git

After you've done that use the next command in the WebApp's directory to run the web app:

python -m flask --app source run --port 8000

And for the last step, open your browser at 127.0.0.1:8000 and there it is!
